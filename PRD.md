## User Workflow

### 5-Step Process

1. **Project Brief**
   - Describe your project in plain English
   - Example: "I want to build a website", "I want a mobile app"

2. **AI Tools Selection**
   - Choose from Claude, OpenAI, and other AI providers
   - Multiple models available for different tasks

3. **Answer Open Questions**
   - AI asks clarifying questions:
     - "How should the notification behave?"
     - "Should users get an onboarding?"
     - "Can a user add friends?"

4. **Project Plan**
   - Comprehensive project plan generated
   - Task breakdown and milestones

5. **Create Docs**
   - Full documentation suite generated
   - Ready for AI coding tools
